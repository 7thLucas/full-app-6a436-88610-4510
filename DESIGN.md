# autoMoM Design Guidelines

## Visual Identity
Dark navy and teal — professional, focused, modern. Feels like a tool built for serious teams, not a toy.

## Color Palette
- Background: deep navy (#0f172a)
- Card surfaces: slightly lighter navy (#1e293b)
- Primary action: teal (#0d9488)
- Primary foreground: white (#ffffff)
- Secondary: slate blue-gray (#334155)
- Accent: cyan-teal (#06b6d4)
- Muted: (#1e293b)
- Muted foreground: (#94a3b8)
- Foreground: (#e2e8f0)
- Border: (#334155)
- Destructive: (#ef4444)
- Sidebar background: (#0f172a)
- Navbar background: (#0f172a)

## Typography
- Headings: Space Grotesk — geometric, modern, distinctive
- Body: Inter — clean, legible, professional

## Elevation & Depth
- Cards use subtle border + slight background lift (not heavy shadows)
- Sidebar darker than main content area
- Use border-border for all dividers

## Components
- Sidebar: dark, compact, icon + label navigation items
- Cards: rounded-lg, border, bg-card
- Buttons: primary uses teal, ghost/outline for secondary actions
- Tables: compact, no zebra stripes, border-bottom per row
- Badges: colored by status (pending = yellow, done = green, in-progress = blue)

## Layout
- Sidebar (left, fixed) + main content area
- Top navbar with breadcrumb + user menu
- Content area: padded, max-width container, card-based sections
- Data-dense but never cluttered — consistent spacing

## Feel
Clean. Focused. Fast. This is a tool that respects the user's time.
